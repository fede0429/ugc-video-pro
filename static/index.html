账号体系

支持管理员与多个子账户密码登录。

部署后建议使用固定 admin 账户进入系统，在管理后台创建子账户。子账户只需邮箱 + 密码即可登录，admin 可以停用/删除其他子账户。

  • admin 为最高管理员，拥有子账户管理能力。
  • 子账户使用邮箱和密码登录，不需要再单独部署第二套系统。
  • 项目 1 / 项目 2 共用同一套账号与权限。
