## 1. 项目概述 / Project Overview

UGC Video Pro 是一个**完整的独立应用程序**，通过 Telegram Bot 界面为电商和营销人员生成高质量 UGC 产品视频。

核心特性：
- 🎬 **多模型支持**：Sora 2/Pro、Veo 3.0/3.1 Pro、Seedance 2.0
- 🔗 **帧链技术**：将多个短片无缝拼接成 15-120 秒的长视频
- 🖼 **产品保真度**：通过 Gemini 图像分析，确保 AI 精确还原产品外观
- 🌍 **多语言支持**：中文 / 英文 / 意大利语界面和旁白
- ☁️ **云端存储**：自动上传到 Google Drive
- 🐳 **容器化部署**：完整 Docker 支持，一键部署到任何 VPS
